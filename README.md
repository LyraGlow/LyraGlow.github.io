# 🌟 Jianhao Huang's Personal Website & Blog

Welcome to the source code repository of my personal website: [LyraGlow.github.io](https://lyraglow.github.io). 

Here, I share my academic research, technical explorations, and notes on software development. 

## 👨‍💻 About Me

I am a student and researcher with a strong passion for **Spatio-temporal Big Data Analysis**, particularly in the maritime domain. My current work focuses on processing AIS (Automatic Identification System) data and exploring vessel trajectory prediction and restoration using deep learning models (such as LSTM, RNN, and Attention mechanisms).

### 🔬 Research & Interests
- **Spatio-temporal Data Mining:** Processing and interpolating massive trajectory datasets.
- **Machine Learning / Deep Learning:** Reproducing and innovating upon time-series prediction models.
- **Big Data Applications:** Applying data science to real-world scenarios, including collegiate competitions like the China Undergraduate Computer Design Contest (4C).

### 🛠 Tech Stack & Tools
- **Languages:** Python, C/C++, Shell
- **OS & Environments:** Linux (Ubuntu 24.04), Customizing workflows (Zotero)
- **Data & Visualization:** Pandas, NumPy, Matplotlib (Developer of the lightweight plotting utility `Drawn`)
- **Writing & Formatting:** Markdown, LaTeX

## 🚀 Projects Highlight

- **Inland Ship Trajectory Restoration:** Reproducing and improving upon research using LSTM, including advanced data processing and timestamp-authenticity checks for AIS records.
- **Drawn:** A customized, lightweight Python class for elegant scientific plotting with Matplotlib.
- **Linux Workspace Optimization:** Continuous improvements in setting up deep learning environments (NVIDIA configurations) and efficient academic reading workflows.
